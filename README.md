# Data-Driven-Trivia-Night-Scoring-System

Simple steps to open the scoring system
1. Find the file in the github
2. Download the file
3. open the downloaded file in your device

**  Boom -- The Trivia Night Scoring System is in your hands Now **

How to use the Scoring System
1. Assign the team names
2. Assign the scores to each team
3. Save changes and end the ongoing round & move to the next one
4. You can find the leaderboard on the widgets
